Python

Python adalah bahasa pemrograman tingkat tinggi yang ditafsirkan, serbaguna, dan banyak digunakan dalam berbagai bidang seperti pengembangan web, analisis data, dan machine learning. Python dikembangkan oleh Guido van Rossum pada tahun 1991 dan dikenal dengan sintaksnya yang mudah dipahami.

JavaScript

JavaScript adalah bahasa pemrograman tingkat tinggi yang digunakan terutama untuk mengembangkan antarmuka web yang interaktif dan dinamis. Bahasa ini juga bisa digunakan untuk membuat aplikasi, tools, dan game di web. JavaScript memungkinkan pengembang untuk menambahkan interaksi dan fungsionalitas pada halaman web tanpa perlu memuat ulang halaman.

Java

Java adalah bahasa pemrograman populer yang multiplatform dan berorientasi-objek. Java dapat digunakan sebagai platform lewat Java virtual machine (JVM), yang dapat diinstal di sebagian besar komputer dan perangkat seluler.

C#

C# adalah bahasa pemrograman berorientasi objek yang dikembangkan oleh Microsoft, merupakan bagian dari platform .NET. C# digunakan untuk mengembangkan berbagai macam aplikasi, seperti aplikasi desktop, aplikasi web, game, dan aplikasi seluler. Bahasa ini dikenal karena mudah dipelajari, serbaguna, dan memiliki komunitas yang besar.

C

C LANG adalah singkatan dari "The C Programming Language", sebuah buku terkenal karya Brian Kernighan dan Dennis Ritchie yang menjadi referensi utama untuk belajar bahasa pemrograman C. Bahasa C sendiri adalah bahasa pemrograman serbaguna yang sangat populer dan sering digunakan dalam pengembangan sistem dan aplikasi

C++

C++ adalah bahasa pemrograman komputer yang berorientasi objek (Object Oriented Programming) dan banyak digunakan untuk pengembangan berbagai jenis aplikasi, termasuk perangkat lunak, game, dan sistem. C++ dikembangkan sebagai perluasan dari bahasa C dan mendukung berbagai gaya pemrograman seperti prosedural, fungsional, dan generik.

Go

Golang, atau Go, adalah bahasa pemrograman open-source yang dikembangkan oleh Google. Bahasa ini dirancang untuk efisien, mudah dipelajari, dan mampu menangani tugas-tugas berat dengan cepat. Golang cocok untuk berbagai aplikasi, termasuk pengembangan aplikasi web, backend, dan sistem yang membutuhkan kinerja tinggi.

R

R LANG atau bahasa R adalah bahasa pemrograman dan perangkat lunak yang digunakan untuk komputasi statistik dan grafik. Bahasa ini populer dalam bidang data science, analisis data, dan machine learning. R dikembangkan oleh Ross Ihaka dan Robert Gentleman dari University of Auckland, dan merupakan pengembangan dari bahasa S.

Swift

Swift adalah bahasa pemrograman yang dikembangkan oleh Apple untuk digunakan dalam pengembangan aplikasi di berbagai platform Apple seperti iOS, macOS, watchOS, dan tvOS. Swift dikenal sebagai bahasa yang modern, aman, dan mudah dipelajari.

PHP

PHP adalah bahasa pemrograman scripting server-side yang digunakan untuk membuat aplikasi web dinamis dan interaktif. PHP dieksekusi di server sebelum halaman web dikirimkan ke browser pengguna. Bahasa ini sangat populer karena kemudahan penggunaannya dan kemampuannya untuk membuat berbagai jenis aplikasi web.
